# ApacheFriends Website

The Apache Friends website is based on [Middleman](https://middlemanapp.com/).

## Requirements

- Docker
- Docker compose

## TL;DR;

```console
$ docker-compose run --rm middleman bundle exec middleman build 
```

The static website is built at the `build/` folder.
